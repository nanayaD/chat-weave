Chat Weave 사용 방법

1. 이 폴더의 index.html 파일을 브라우저로 엽니다.
2. X 또는 Discord 대화 로그를 붙여넣고 화자 정보를 입력합니다.
3. 파싱하기를 눌러 미리보기를 확인합니다.
4. 내보내기 메뉴에서 HTML 파일 또는 티스토리 코드를 생성합니다.

주의
- 이 앱은 서버에 데이터를 업로드하지 않습니다.
- 브라우저 안에서만 동작합니다.
- 작업을 나중에 이어서 편집하려면 내보내기 > 작업 JSON 저장을 사용하세요.
- 작업 JSON은 완성본이 아니라 이 앱에서 다시 불러오기 위한 편집용 파일입니다.
