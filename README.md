# Chat Weave

복사해 온 X 타래 또는 Discord 대화 텍스트를 로컬 브라우저에서 파싱하고, 보기 좋은 세로형 대화 백업본으로 내보내는 React + TypeScript 웹앱입니다. 서버 업로드 없이 브라우저 안에서만 작업합니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

브라우저에서 표시되는 로컬 주소를 엽니다. 기본 개발 주소는 보통 `http://localhost:5173`입니다.

## 사용 흐름

1. X 또는 Discord 플랫폼을 선택합니다.
2. 왼쪽 패널에서 화자를 등록합니다.
3. 오른쪽 원문 영역에 복사한 대화 텍스트를 붙여넣습니다.
4. `파싱하기`를 눌러 가운데 미리보기를 만듭니다.
5. 표시 정보, 익명 처리, 연속 발화 처리, 스택형/좌우형 테마, 다크 모드, 글자 크기, 말풍선 폭을 조정합니다.
6. 불확실 항목은 미리보기와 목록에서 확인하고, 원문/앞뒤 대화 위치로 바로 이동할 수 있습니다.
7. HTML 파일, 티스토리 코드, 프로젝트 JSON으로 내보냅니다.

## 현재 파서의 한계

- 플랫폼에 로그인하거나 API를 호출하지 않습니다.
- 사용자가 붙여넣은 텍스트만 처리합니다.
- 시간은 해석하지 않고 표시만 합니다.
- 연속 발화 병합은 인접한 같은 화자 기준으로만 동작합니다.
- 등록되지 않은 화자나 애매한 줄은 제외/불확실 항목으로 표시됩니다.
- X와 Discord 복사본의 실제 형태가 다양하므로 일부 줄은 직접 정리해야 할 수 있습니다.
